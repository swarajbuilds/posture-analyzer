Posture Analyzer

An AI-powered web application that detects and analyzes user posture in real time using Python, OpenCV, and MediaPipe.

Features

- Real-time posture detection
- Exercise monitoring
- Dashboard for posture analysis
- Posture history and progress tracking
- Alerts for incorrect posture
- User authentication

Technologies Used

- Python
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript

Live Demo

Website: https://posture-ai-frontend-z6okzblb5a-uc.a.run.app/dashboard.html

Project Status

This project was developed as a student project to promote healthy posture through AI-powered posture analysis.
